# 4.Projekt

Custom Next.js website for selling original handmade paintings online with your own checkout, shipping, and admin flow.

## Features

- Public storefront for artworks
- Product pages with details and images
- Cart and checkout foundation
- Admin-ready structure for managing products and orders

## Getting Started

1. Install dependencies:
   `npm install`
2. Start development server:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## Controls

- `npm run dev`: Start local development server
- `npm run build`: Build production bundle
- `npm run start`: Run production server
- `npm run lint`: Run ESLint checks
